# 🚀 LinkLoot | Deployment Guide

To get this website live on the internet for free, follow these exact steps:

### Step 1: Create a GitHub Repository
1. Log in to [GitHub.com](https://github.com).
2. Click the **"New"** button (green) or the **"+"** icon in the top right and select **"New repository"**.
3. Name it `link-loot`.
4. Keep it **Public** and click the green **"Create repository"** button.

### Step 2: Upload Your Files
1. On your new repository page, click the link that says **"uploading an existing file"** (it's in the setup box).
2. Drag and drop your project files (like `src`, `package.json`, etc.) from your computer into the GitHub window.
3. Click the green **"Commit changes"** button at the bottom.

### Step 3: Go Live with Firebase
1. Go to [console.firebase.google.com](https://console.firebase.google.com/).
2. Click **"Add Project"**, name it `LinkLoot`, and finish the setup.
3. Click the **Hamburger Menu (three lines)** in the top left.
4. Click **"Build"** (the hammer icon).
5. Click **"App Hosting"**.
6. Connect your GitHub account, select your `link-loot` repo, and click **"Deploy"**.

**Wait 2-3 minutes, and Firebase will give you a live link!**
